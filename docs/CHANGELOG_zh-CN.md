# 中文更新日志

## 2026-05-25 - LBTV 分叉功能恢复版

本版本基于上游 `CS2-Bot-Improver` 最新整包重新构建，恢复并整合了 LBTV 分叉中的人机增强功能。

### 新增与恢复

- 恢复 BOT 击杀嘲讽功能。
- 恢复 BOT 开局嘲讽、连杀嘲讽、残局/保枪场景嘲讽和 MVP 嘲讽。
- 恢复 BOT AI 聊天回复功能，可通过配置文件修改 API 地址和密钥。
- 恢复 `lbtv_bot_taunt 0/1` 开关。
- 恢复 `lbtv_bot_chat 0/1` 开关。
- 恢复 `lbtv_difficulty` 难度显示命令。
- 回合开始时会向真人玩家显示当前 BOT 难度。
- 恢复回合结束伤害统计，并优化道具伤害统计。

### 道具与伤害统计优化

- 火焰、手雷和道具直接砸到人身上的伤害现在会计入回合伤害统计。
- BOT 生成的烟、闪、雷、火会写入投掷者、原始投掷者、拥有者、初始位置、初始速度和弹性参数，提升道具来源识别准确性。
- 回合内团队道具数量增加硬上限：
  - 闪光弹：10 个
  - 烟雾弹：5 个
  - 手雷：5 个
  - 燃烧弹：5 个
- 回合内单个 BOT 道具数量增加硬上限：
  - 闪光弹：2 个
  - 烟雾弹：1 个
  - 手雷：1 个
  - 燃烧弹：1 个
- BOT 在购买时间结束后会记录本回合经济预算，本回合自动投掷道具的花费不能超过该预算。
- BOT 杀人后获得的额外经济不会继续被插件用于本回合购买更多道具。

### 平衡性调整

- 降低 BOT 受伤后临时穿烟可视能力。
- `bot_max_visible_smoke_length` 的临时增强距离从 `4000` 降为 `1000`。
- 临时增强持续时间从 `1.0` 秒降为 `0.35` 秒。
- 拆包烟相关的临时穿烟增强也使用同样的较低持续时间。
- BOT 使用插件生成的道具后会进入短暂禁止开枪硬直，避免瞬发道具没有代价。

### 配置文件

BOT AI 聊天配置文件路径：

```text
game/csgo/addons/counterstrikesharp/configs/plugins/BotTaunt/BotTaunt.json
```

常用字段：

```json
{
  "AiChatEnabled": true,
  "AiApiUrl": "https://your-domain.com/api/chat",
  "AiApiKey": "your_api_key_here"
}
```

修改后需要重启 CS2 或服务器。
